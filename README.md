# Pokemon App 🔥

แอป React Native ที่ดึงข้อมูลจาก PokeAPI เพื่อแสดงรายชื่อและรายละเอียดของ Pokémon

---

## 📱 Features

- 🔍 ค้นหา Pokémon ได้ทั้งหมด
- 📄 แสดงรายละเอียด Pokémon
- 🧬 Evolution Chain
- 📊 Stats (HP / Attack / Defense)
- 🖼️ รูปภาพ Pokémon
- 📑 แบ่งหน้า (Pagination)

---

## 🛠️ Tech Stack

- React Native
- Expo
- Expo Router
- TypeScript
- PokeAPI
